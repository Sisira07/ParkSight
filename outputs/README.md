# Outputs Directory

Generated by `python -m parksight.cli`. Not committed to version control
(see `.gitignore`) — this directory is regenerated from your own input CSV.

## Artifacts produced by one pipeline run

| File | Produced by | Required? |
|------|-------------|-----------|
| `hotspot_clusters.csv` | `hotspot_detection.py` | Yes — the backend will not start serving data without this |
| `nonjunction_hotspots.csv` | `nonjunction_hotspots.py` | Optional |
| `risk_scores.csv` | `risk_scoring.py` | Optional |
| `spatial_stability.json` | `spatial_validation.py` | Optional |
| `traffic_enrichment.csv` | `traffic_enrichment.py` | Optional — requires `GOOGLE_MAPS_API_KEY` for real probes |
| `capacity_impact.csv` | `capacity_impact.py` | Optional |
| `patrol_plan.csv` | `patrol_optimization.py` | Optional |
| `temporal_audit.json` | `temporal_audit.py` | Optional |
| `bias_audit.json` | `bias_audit.py` | Optional |
| `evidence_ledger.csv` | `evidence.py` | Optional |
| `executive_report.md` | `reporting.py` | Optional |
| `dbscan_meta.json` | `hotspot_detection.py` | Optional sidecar |
| `monthly_panel.csv` | `temporal_audit.py` | Optional sidecar |

The backend (`backend/app/services/artifact_loader.py`) reads whichever of
these are present and reports the rest as unavailable — it never fabricates
a replacement value for a missing artifact.

## Multiple runs

If `PARKSIGHT_OUTPUT_DIR` contains timestamped subdirectories instead of
artifacts directly, the backend automatically selects the most recently
modified one, or a specific one via `PARKSIGHT_RUN_ID`.
